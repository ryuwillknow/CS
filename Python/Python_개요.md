## 식별자

변수의 이름은 식별자라고 한다. 읽기 쉽고 이해하기 쉬워야하기 때문이다.

변수명을 많이 지어봐야한다. 규칙은 조금 필요하다

1. 영문알파벳, 언더스코어, 숫자로 구성
2. 식별자의 첫글자는 숫자가 올 수 없음
3. 대소문자 구분한다.
4. 다음 키워드는 사용하면 안된다.
5. 내장함수, 모듈의 이름은 사용하면 안된다.
