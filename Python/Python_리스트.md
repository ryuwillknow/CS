# 리스트

- **정말 너무 싫은 경우가 있다**.

```python
#리스트 안에 리스트가 있을 때 전체가 더해진다.

a = [[1,2],[2,3],[3,4],[4,5]]
a[1] = a[1]+[2] #or a[1] += [2]
#이거 정상임

x = [[]] * 5 #이게 비정상 연동된다.
#그러면 어떻게 해야하는데 이렇게
# 문제는 [[]]*3 하고 x[1][2] 할때 생긴다
[[0]*100 for i in range(100))
```

이거 아주 악질임

# List comprehension
```python
a = [x**3 for x in range(1,10) if x%2==0]
```

# 리스트 복사
순서 없는 Set이나 Dict에 대해서

얕은 복사와 깊은 복사

s.copy() : 얕은 복사본을 반환

dict.get(key[,default})

default는 선택적인 값이다. 기본적으로 None이다 그것을 바꿔줄 수 있다.

```python
a = [1,2,['a','b']]
```

‘a’ 와 ‘b’의 위치를 따로 저장하고 그 주소를 리스트에 저장한다. 

a[2][0] 이면 모든 ‘a’가 0으로 바뀐다.

copy안에 deepcopy를 사용한다. 

copy.copy 는 얕은 복사

**리스트 복사는 프린트 꼭 해봐야한다.**
