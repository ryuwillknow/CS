집합 미친듯

교집합, 차집합 가능하다. 

a - b 차집합   - 더 커도 된다. 그냥 빼준다.

a | b 합집합

a & b 교집합

더하기는 안된다.

add remove 가능

조심할 거 

```python
a = {}  # 이건 dict이다.
a = set() #이렇게 정의해야한다.
```
